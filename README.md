# GoogleCalendar
A simple Electron Wrapped Google Calendar Application
## Install

```sh
git clone https://github.com/andrepolischuk/keep
cd keep
npm install
```

## Run

```sh
npm start
```

## Build

Build the application for specified platform:

```sh
npm run build:macos
npm run build:linux
npm run build:windows
```
